Realizei pesquisas para cores e tipografias, criei logotipo e o design no Figma.  Encarreguei-me de conceber a experiência do usuário (UX) e a interface do usuário (UI) 
de forma impactante. Desenvolvi a codificação e implementação do site utilizando HTML, CSS, BOOTSTRAP e JAVASCRIP, garantindo uma presença online bem-sucedida, 
para um escritório de advocacia. O objetivo era atrair e engajar potenciais clientes, expor as áreas trabalhadas, informações sobre o escritório e um formulário
para possíveis dúvidas. Posteriormente, executei o deploy para tornar o site acessível ao público na plataforma de hospedagem que eu tenho e o domínio obtido pelo cliente.
